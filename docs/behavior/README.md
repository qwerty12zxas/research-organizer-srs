# Диаграммы поведения системы — Органайзер научных статей

## Use Case
См. `../use_case/use_case_diagram.png`.

**Актор:** Пользователь (основной пользователь системы)

**Основные Use Cases:**
- Добавление/импорт статьи (включая валидацию)
- Просмотр, поиск и фильтрация
- Управление статусом чтения
- Ежедневный обзор/аналитика

## Диаграммы последовательности
1. **Добавление статьи** — `sequence/add_article_sequence.puml` 
2. **Отметка "прочитано"** — `sequence/mark_read_sequence.puml`
3. **Фильтрация статей** — `sequence/filter_articles_sequence.puml`
4. **Ежедневный workflow** — `sequence/daily_workflow_sequence.puml`

## Инструменты
- **PlantUML** — код в *.puml, экспорт PNG/SVG
